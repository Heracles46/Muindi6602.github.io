Muindi WhatsApp Bot MD Deployment

⚙️ Setup.

  1. Click SCAN and scan the QR code through the "WhatsApp Linked Devices" option in your WhatsApp app.

  2. You will get a session ID in WhatsApp. (Copy the Session ID only).

1. Deploy on Heroku

  1. If you don't have an account on Heroku, create an account now: Heroku.

  2. If you don't have a GitHub account, sign up now: GitHub.

  3. FORK this repository. (Copy the forked repository link as you'll use it while deploying).

  4. Now DEPLOY.

2. Deploy on Koyeb.

  1. You can sign up in KOYEB.

  2. Get your Database URL in DATABASE_URL. (Copy your Database URL).

  3. Get your Koyeb API Key Here (Copy your API Key).

  4. Now DEPLOY.

3. Deploy on Ubuntu (Termux):

    1. First above all, install Termux Apk into your android phone in the official store. click Here to download.

    2. Run this command in the Termux App to update and upgrade to-date-files.
    📋 Copy 
    apt-get update && apt-get upgrade -y

    3. Install wget proot and git:
    📋 Copy 
    apt-get install wget proot git -y

    4. Go to HOME folder in Termux:
    📋 Copy 
    cd

    5. Fetch the script from the github:
    📋 Copy 
    git clone https://github.com/MFDGaming/ubuntu-in-termux.git

    6. Go to the script directory:
    📋 Copy 
    cd ubuntu-in-termux

    7. Give execution permission to the script:
    📋 Copy 
    chmod +x ubuntu.sh

    8. Execute the script: (Try any between the two codes. on will work)
    📋 Copy 
    ./ubuntu.sh -y
    OR
    📋 Copy 
    /ubuntu.sh -y

    9. start ubuntu:
    📋 Copy 
    ./startubuntu.sh

    10. Install Neofetch in Ubuntu shell:
    📋 Copy 
    apt-get install neofetch

    11. Run the update again to activate new ubuntu files:
    📋 Copy 
    apt -y update && apt -y upgrade

    12. Install git,ffmpeg, and curl
    📋 Copy 
    apt -y install git ffmpeg curl

    13. Install node.js:
    📋 Copy 
    curl -fsSl https://deb.nodesource.com/setup_lts.x | bash - && apt -y install nodejs

    14. Install yarn:
    📋 Copy 
    npm install -g yarn

    15. Install pm2
    📋 Copy 
    yarn global add pm2

    16. Clone the repository:
    📋 Copy 
    git clone https://github.com/lyfe00011/whatsapp-bot-md YourbotName cd YourbotName

    17. Install packages and network concurrency:
    📋 Copy 
    yarn install --network-concurrency 1

    18. Enter Environment Variables: Copy-paste the lines below and replace the variable values with yours:
    📋 Copy 
    echo "SESSION_ID = Session_Id
            PREFIX = .
            STICKER_PACKNAME = MuindiBot
            ALWAYS_ONLINE = true
            RMBG_KEY = null
            LANGUAG = en
            WARN_LIMIT = 1
            FORCE_LOGOUT = false
            BRAINSHOP = 159501,6pq8dPiYt7PdqHz3
            MAX_UPLOAD = 200
            REJECT_CALL = true
            SUDO = 254115783375
            TZ = Africa/Nairobi
            VPS = true
            AUTO_STATUS_VIEW = no-dl
            SEND_READ = true
            AJOIN = true
            DISABLE_START_MESSAGE = false
            PERSONAL_MESSAGE = null" > config.env









    19. Save the above by clicking Ctrl + O, them press Enter.

    20. Start the bot:
    📋 Copy 
    pm2 start . --name YourbotName --attach --time

    21. In Progress 20 above, you can replace ( pm2 start . --name botName --attach --time) with this to prevent the bot from stopping even after closing the app:
    📋 Copy 
    nohup pm2 start . --name YourbotName --attach --time
